## 3. Create the Database Table
We need a table to store the scam reports.
1.  Go to the **SQL Editor** (Bar icon on the left sidebar).
2.  Click **"New Query"**.
3.  Paste this exact code and click **RUN**:

```sql
create table reports (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  phone_number text,
  scam_type text,
  risk_score int,
  transcript text
);
```
